# ChatGPT Math Dir Resolver

A browser extension that fixes the text direction of mathematical formulas (KaTeX elements) on ChatGPT by setting `dir="ltr"`.

## Installation

### Chrome / Edge / Brave / Opera

1. Download the latest release: [chatgpt-math-dir-resolver)-v1.0.zip](https://github.com/miladniroee/chatgpt-math-dir-resolver/releases)
2. Unzip the file
3. Open your browser and go to:
   - **Chrome**: `chrome://extensions/`
   - **Edge**: `edge://extensions/`
   - **Brave**: `brave://extensions/`
   - **Opera**: `opera://extensions/`
4. Enable **Developer mode** (toggle in top-right corner)
5. Click **Load unpacked**
6. Select the unzipped folder

### Firefox

1. Download the latest release: [chatgpt-math-dir-resolver)-v1.0.xpi](https://github.com/miladniroee/chatgpt-math-dir-resolver/releases)
2. Open Firefox and go to `about:addons`
3. Click the gear icon ⚙️ > **Install Add-on From File**
4. Select the downloaded `.xpi` file

## Usage

Once installed, the extension automatically works on ChatGPT. All mathematical formulas will display with left-to-right direction.
